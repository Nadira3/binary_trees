# 0x1C.c

Functions to print binary trees in a pretty way

```
                           .----------------------(006)-------.
                      .--(001)-------.                   .--(008)--.
                 .--(-02)       .--(003)-------.       (007)     (009)
       .-------(-06)          (002)       .--(005)
  .--(-08)--.                           (004)
(-09)     (-07)
```

Programs are compiled using the following flags
```
gcc -Wall -Wextra -Werror -pedantic binary_tree_print.c...
```
